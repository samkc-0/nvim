# 🌀 minimal neovim config

clean, modular neovim setup built from scratch. plugins are opt-in, filetype-specific configs live in `ftplugin/`, and nothing loads unless it needs to.

---

## 🛠️ install

### clone into nvim config directory

```sh
cd ~/.config/
git clone https://github.com/samkc-0/nvim
```
